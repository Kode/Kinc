Detailed Description

Audio1 is a high-level audio-API that lets you directly play audio-files. Depending on the target-system it either sits directly on a high-level system audio-API or is implemented based on Audio2.

◆ kinc_a1_mix()

KINC_FUNC void kinc_a1_mix ( kinc_a2_buffer_t *  buffer,
uint32_t  samples 
)

Mixes audio into the provided buffer. kinc_a1_init sets this as the callback for a2 but you can also call it manually to mix a1-audio with your own audio. To do that, first call kinc_a1_init, then call kinc_a2_set_callback to set it to your own callback and call kinc_a1_mix from within that callback. Please be aware that the callback will run in a separate audio-thread.

Parameters
bufferThe audio-buffer to be filled
samplesThe number of samples to be filled in
