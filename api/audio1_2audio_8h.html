Detailed Description

Audio1 is a high-level audio-API that lets you directly play audio-files. Depending on the target-system it either sits directly on a high-level system audio-API or is implemented based on Audio2.

Mixes audio into the current buffer. Normally this is called internally but you can also call it manually to mix a1-audio with your own audio. To do that first call kinc_a1_init, then call kinc_a2_set_callback to set it to your own callback and call kinc_a1_mix from that callback.

Parameters
bufferThe audio-buffer to be filled
samplesThe number of samples to be filled in
