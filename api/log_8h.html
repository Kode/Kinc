Detailed Description

Contains basic logging functionality.

Logging functionality is similar to plain printf but provides some system-specific bonuses.

Logging function similar to printf including some system-specific bonuses.

On most systems this is equivalent to printf. On Windows it works with utf-8 strings (like printf does on any other target) and also prints to the debug console in IDEs. On Android this uses the android logging functions and also passes the logging level.

log_levelThe logLevel is ignored on all targets but Android where it is converted to the equivalent Android log level
