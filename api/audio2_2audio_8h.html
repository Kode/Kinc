Detailed Description

Audio2 is a low-level audio-API that allows you to directly provide a stream of audio-samples.

void *  userdata 
)

Sets the callback that's used to provide audio-samples. This is the primary method of operation for Audio2. The callback is expected to write the requested number of samples into the ring-buffer. The callback is typically called from the system's audio-thread to minimize audio-latency.
