file  audio.h [code]
 Audio1 is a high-level audio-API that lets you directly play audio-files. Depending on the target-system it either sits directly on a high-level system audio-API or is implemented based on Audio2.
 
file  sound.h [code]
 Sounds are pre-decoded on load and therefore primarily useful for playing sound-effects.
 
file  soundstream.h [code]
 Sound-Streams are decoded while playing and as such are useful for large audio-files like music or speach.
